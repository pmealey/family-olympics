# Family Olympics - UI Specification

## Design System

### Theme: Winter Wonderland

A crisp, modern winter theme with bold team color accents.

### Color Palette

```css
:root {
  /* Base - Winter Blues & Whites */
  --color-bg-primary: #f0f7ff;        /* Light ice blue background */
  --color-bg-secondary: #ffffff;       /* Pure white cards */
  --color-bg-dark: #1a2b3c;           /* Dark navy for contrast */
  
  /* Text */
  --color-text-primary: #1a2b3c;      /* Dark navy */
  --color-text-secondary: #5a6b7c;    /* Muted blue-gray */
  --color-text-inverse: #ffffff;      /* White on dark */
  
  /* Accent - Ice Blue */
  --color-accent: #3b82f6;            /* Bright winter blue */
  --color-accent-light: #93c5fd;      /* Light accent */
  --color-accent-dark: #1d4ed8;       /* Dark accent */
  
  /* Team Colors */
  --color-team-green: #22c55e;
  --color-team-green-light: #86efac;
  --color-team-green-dark: #15803d;
  
  --color-team-pink: #ec4899;
  --color-team-pink-light: #f9a8d4;
  --color-team-pink-dark: #be185d;
  
  --color-team-yellow: #eab308;
```

### Typography

```css
:root {
  /* Font Families */
  --font-display: 'Outfit', sans-serif;      /* Headers, bold text */
  --font-body: 'Inter', sans-serif;          /* Body text */
  --font-mono: 'JetBrains Mono', monospace;  /* Scores, times */
  
  /* Font Sizes */
  --text-xs: 0.75rem;
  --text-sm: 0.875rem;
  --text-base: 1rem;
  --text-lg: 1.125rem;
  --text-xl: 1.25rem;
  --text-2xl: 1.5rem;
  --text-3xl: 2rem;
  --text-4xl: 2.5rem;
}
```

### Responsive Breakpoints

```css
/* Mobile-first approach */
/* Default: Mobile (< 640px) */
@media (min-width: 640px)  { /* sm: Tablet */ }
@media (min-width: 1024px) { /* lg: Desktop */ }
```

1. Minimum touch target: 44x44px

## Loading States

1. **Initial load:** Full-page skeleton
2. **Data fetching:** Inline spinners
3. **Form submission:** Button loading state
4. **Optimistic updates:** Immediate UI change, rollback on error

1. **No events:** "No events scheduled yet. Check back soon!"
2. **No scores:** "Results will appear here once the event is complete."
3. **No teams:** "Teams haven't been set up yet."

## Error States

1. **Network error:** Toast notification with retry option
2. **Validation error:** Inline field errors
3. **Not found:** Friendly 404 page with navigation home
